# The Blueprint — Purpose Discovery Website

A fully AI-powered purpose discovery assessment. Users answer 5 questions and receive a personalised Blueprint report generated by Claude AI.

---

## Folder Structure

```
blueprint/
├── api/
│   └── analyze.js       ← Serverless function (AI logic lives here)
├── public/
│   └── index.html       ← The entire website
├── vercel.json          ← Vercel routing config
├── package.json         ← Project info
└── README.md            ← This file
```

---

## How to Deploy (Step by Step)

### Step 1 — Get a FREE Anthropic API Key
1. Go to **https://console.anthropic.com**
2. Sign up for a free account
3. Click **"API Keys"** in the left sidebar
4. Click **"Create Key"** → copy the key (starts with `sk-ant-...`)
5. Keep it safe — you won't see it again

---

### Step 2 — Upload to GitHub
1. Go to **https://github.com** and sign up / log in
2. Click the **"+"** icon → **"New repository"**
3. Name it `blueprint` → click **"Create repository"**
4. Upload all files from this ZIP — drag and drop the entire folder contents
5. Click **"Commit changes"**

---

### Step 3 — Deploy on Vercel
1. Go to **https://vercel.com** and sign up with your GitHub account
2. Click **"Add New Project"**
3. Click **"Import"** next to your `blueprint` repository
4. Leave all settings as default — click **"Deploy"**
5. Wait ~30 seconds for it to build ✅

---

### Step 4 — Add Your API Key (IMPORTANT)
Without this step the AI won't work.

1. In your Vercel dashboard, click on your **blueprint** project
2. Go to **Settings** → **Environment Variables**
3. Click **"Add New"**
4. Set:
   - **Name:** `ANTHROPIC_API_KEY`
   - **Value:** paste your key from Step 1 (e.g. `sk-ant-api03-...`)
   - **Environment:** check All (Production, Preview, Development)
5. Click **"Save"**
6. Go to **Deployments** → click the three dots on your latest deployment → **"Redeploy"**

---

### Step 5 — Your site is live! 🎉
Vercel gives you a free URL like:
`https://blueprint-yourname.vercel.app`

Share it with anyone. It works on mobile and desktop.

## How It Works

```
User answers 5 questions
        ↓
Website sends answers to /api/analyze
        ↓
Vercel serverless function calls Claude AI
(API key is kept secret on the server)
        ↓
Claude analyses the answers and returns a JSON report
        ↓
Website displays the personalised Blueprint
```

---

## Notes
- The API key is **never exposed** in the browser — it only lives on Vercel's servers
- Anthropic's free tier gives you enough credits to test the app
- Users can save their report as PDF using the "Save as PDF" button (uses browser print)
- Voice recording works on Chrome and Edge browsers
